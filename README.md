# SaaS Churn MLOps

End-to-end churn prediction project with a training pipeline, MLflow experiment tracking, and a FastAPI inference API (Docker-ready).

## Goals
- Train and evaluate churn models (baseline + boosted model)
- Track experiments with MLflow
- Serve predictions via FastAPI (`/predict`)
- Keep production-style structure: tests, logging, Docker, reproducible commands

## Status
Scaffolding initialized. Implementation in progress.
